How It Works

This application uses LSTM (Long Short-Term Memory) neural networks to predict stock prices based on historical data. The model analyzes patterns in past price movements to forecast future trends.

Technology Stack

  • Backend: Python Flask
  • Machine Learning: TensorFlow/Keras LSTM
  • Data Source: Yahoo Finance API
  • Frontend: HTML, CSS, JavaScript

Model Process

  1. Fetch historical stock data (1-5 years)
  2. Preprocess and normalize the data
  3. Create sequences for time series prediction
  4. Train LSTM neural network
  5. Generate predictions for future prices
